Projetos em Dados e Automação

RPA para Preenchimento de Formulário

O projeto visa automatizar o processo de preenchimento dos dados no formulario de cadastro de produtos, utilizando Python e interagindo com a interface gráfica do usuário. Os dados dos produtos são lidos a partir de um arquivo CSV, e as ações de preenchimento dos formulários ou campos necessários são automatizadas. Isso é realizado para economizar tempo e minimizar erros humanos durante o processo de cadastro.

As ferramentas utilizadas foram:

  • Python
  • Pandas
  • PyAutoGUI
